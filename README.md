# French Study A1

Локальный бесплатный стартовый веб-кабинет по французскому: разговорные уроки, проверяемые упражнения, произношение, запись голоса и встроенное интервальное повторение FSRS.

Текущий набор — **полный практический A1 для самостоятельной студенческой жизни**: 40 разговорных уроков, 124 упражнения, локальная запись устных заданий и финальный checkpoint по семи навыкам. Дополнительно есть независимый тренажёр чтения: 8 последовательных уроков от коротких буквосочетаний до стыков слов и 127 собственных FSRS-карточек во вкладке «Повторение чтения». Из них 120 атомарных карточек проверяют ровно одно написание в одном контексте, ещё 7 собирают изученные правила в целые слова. Это не официальный сертификат CEFR/DELF; A2/B1/B2 остаются планом масштабирования в `courseRoadmap`.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
.venv/bin/python server.py
```

Открой:

```text
http://localhost:5173
```

`localhost` нужен, чтобы браузер разрешил запись с микрофона. Прогресс, ответы, свои слова, расписание FSRS и голосовые записи хранятся в едином файле `user-data/french-study-data.json` внутри папки проекта. Он общий для всех браузеров и портов на этом компьютере; IndexedDB используется только как локальный кэш.

Распознавание речи тоже локальное: сначала нажми «Записать» → «Стоп», затем «Распознать запись». Сервер передаёт сохранённый аудиофайл установленному на компьютере Whisper, а не облачному STT браузера. Если Whisper ещё не установлен, выполни `.venv/bin/python -m pip install openai-whisper` и убедись, что установлен `ffmpeg`; при первом использовании модели `base` может потребоваться разовая загрузка около 150 МБ. После этого распознавание работает без интернета. При смене кода сервера перезапусти `.venv/bin/python server.py`.

Озвучка использует бесплатные neural-голоса через `edge-tts`. Выбранный голос применяется и к своим словам, и к фразам карточек; встроенный запасной голос не подменяет Denise мужским голосом.

При первом запуске после обновления приложение автоматически переносит данные текущего браузера в общий файл. После этого `localhost:5173`, `127.0.0.1:5173` и другие локальные порты используют один источник прогресса. Для файлового сохранения запускай именно `server.py`, а не `python3 -m http.server`.

## Структура

- `index.html` - оболочка приложения.
- `styles.css` - интерфейс.
- `app.js` - интерфейс кабинета, запись голоса и учебные сценарии.
- `storage.js` - синхронизация общего файла, IndexedDB-кэш, миграция старого прогресса и JSON backup/restore.
- `server.py` - локальный сервер: отдаёт статику, атомарно сохраняет общий прогресс и обслуживает локальные TTS/STT.
- `user-data/french-study-data.json` - общий прогресс всего компьютера; создаётся автоматически и не попадает в git.
- `user-data/french-study-data.backup.json` - предыдущая версия общего файла для аварийного восстановления.
- `tts.js` - озвучка на фронтенде: заранее сгенерированные файлы урока, кэш в IndexedDB для остального, откат на браузерный голос при недоступности сервера.
- `scripts/prewarm_tts.py` - разовая генерация `data/audio/*.mp3` и `data/audio/manifest.json` из основного курса и тренажёра чтения.
- `data/audio/` - заранее озвученные фразы уроков (голос Denise, коммитится в git).
- `cards.js` - словарные, двусторонние, фразовые и cloze-карточки.
- `pronunciation-course.js` - независимая валидация, построение и фильтрация карточек правил чтения.
- `srs.js` - локальная обёртка над FSRS.
- `course-schema.js` и `course-validator.js` - schema/validator курса, стабильных ссылок и roadmap.
- `exercises.js` - нормализация и проверка ответов.
- `mastery.js` - правила prerequisites, готовности урока и допуска к завершению.
- `anki.js` - чистая генерация TSV для Anki.
- `data/lessons.json` - уроки (starter + практический A1, 40 уроков), грамматика, произношение, словарь, открытые ресурсы и roadmap A2-B2.
- `data/pronunciation-course.json` - 8 уроков чтения: буквосочетание → звук → слово → стык слов, 120 атомарных и 7 сборочных карточек повторения.
- `vendor/ts-fsrs` - зафиксированная MIT-сборка `ts-fsrs` 5.4.1 для офлайн-работы.

## Как добавлять уроки

Добавь объект в `data/lessons.json` в массив `lessons`.

Минимальный урок должен содержать:

- `id`, `level`, `title`, `goal`, `scenario`
- `targetPhrase`
- `pronunciationTopic`
- `grammarTopic`
- `dialogue`
- `vocabulary`
- `exercises`
- `cards`

Все словарные записи автоматически превращаются в двусторонние карточки после знакомства с соответствующим уроком. Полные реплики диалогов длиннее двух слов также становятся парой карточек и появляются в отдельном разделе «Фразы» вместе с соответствующим уроком. Свои слова добавляются в разделе `Словарь` и доступны сразу. Anki TSV остаётся необязательным экспортом в настройках.

## Локальные данные

- Обычная перезагрузка, смена браузера, адреса или порта не удаляет и не разделяет прогресс.
- Основной файл обновляется атомарно; перед записью прежняя версия сохраняется как `french-study-data.backup.json`.
- Лёгкая JSON-копия включает всё, кроме аудио.
- Полная JSON-копия включает последние голосовые записи.
- Встроенные слова можно скрывать, но нельзя случайно изменить или удалить.
