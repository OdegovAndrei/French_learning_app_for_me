# French Study Starter

Локальный бесплатный стартовый веб-кабинет по французскому: разговорные уроки, проверяемые упражнения, произношение, запись голоса и встроенное интервальное повторение FSRS.

Текущий набор — **Starter**, а не полный курс до уровня B2. Архитектура поддерживает уровни и модули A0-B2, а в `data/lessons.json` есть `courseRoadmap` для A1/A2/B1/B2. Roadmap — это проверяемый план масштабирования, не заявление о готовом уровне: достижение конкретного уровня допустимо только после наполнения уроками, evidence по всем skill axes и отдельной проверки всей программы по CEFR/DELF.

## Запуск

```bash
python3 -m http.server 5173
```

Открой:

```text
http://localhost:5173
```

`localhost` нужен, чтобы браузер разрешил запись с микрофона. Прогресс, ответы, свои слова, расписание FSRS и последние голосовые записи хранятся локально в IndexedDB.

Важно открывать кабинет по одному и тому же адресу. `localhost:5173` и `127.0.0.1:5173` считаются браузером разными локальными хранилищами.

## Структура

- `index.html` - оболочка приложения.
- `styles.css` - интерфейс.
- `app.js` - интерфейс кабинета, запись голоса и учебные сценарии.
- `storage.js` - IndexedDB, миграция старого прогресса и JSON backup/restore.
- `cards.js` - словарные, двусторонние, фразовые и cloze-карточки.
- `srs.js` - локальная обёртка над FSRS.
- `course-schema.js` и `course-validator.js` - schema/validator курса, стабильных ссылок и roadmap.
- `exercises.js` - нормализация и проверка ответов.
- `mastery.js` - правила prerequisites, готовности урока и допуска к завершению.
- `anki.js` - чистая генерация TSV для Anki.
- `data/lessons.json` - уроки, грамматика, произношение, словарь, открытые ресурсы и roadmap A1-B2.
- `vendor/ts-fsrs` - зафиксированная MIT-сборка `ts-fsrs` 5.4.1 для офлайн-работы.

## Как добавлять уроки

Добавь объект в `data/lessons.json` в массив `lessons`.

Минимальный урок должен содержать:

- `id`, `level`, `title`, `goal`, `scenario`
- `targetPhrase`
- `pronunciationTopic`
- `grammarTopic`
- `dialogue`
- `vocabulary`
- `exercises`
- `cards`

Все словарные записи автоматически превращаются в двусторонние карточки после знакомства с соответствующим уроком. Свои слова добавляются в разделе `Словарь` и доступны сразу. Anki TSV остаётся необязательным экспортом в настройках.

## Локальные данные

- Обычная перезагрузка не удаляет прогресс.
- Лёгкая JSON-копия включает всё, кроме аудио.
- Полная JSON-копия включает последние голосовые записи.
- Встроенные слова можно скрывать, но нельзя случайно изменить или удалить.

## Бесплатные источники

В кабинете заложены ссылки на открытые ресурсы: Tex's French Grammar, TV5MONDE, RFI, Tatoeba, Wiktionary, Lingua Libre, Anki и Praat. Платные курсы и подписки не используются.
